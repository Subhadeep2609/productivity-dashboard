# Productivity Dashboard

A small personal dashboard (HTML/CSS/JS) that shows a weather panel, time/date, quick productivity tools (todo, planner, goals, pomodoro, quotes) and a dynamic background that switches between day and night video assets.

## Features

- Time-based background: uses `assets/day.mp4` and `assets/night.mp4` and switches automatically by local time.
- Top header with theme toggle and GitHub link.
- Right-side compact circular feature cluster (hidden when a tool is opened).
- Tools: Todo list, Planner, Goals, Pomodoro, Quotes.
- Weather and date/time panel with refresh.

## Files

- `index.html` — main markup.
- `style.css` — project styles.
- `script.js` — app logic (theme, clock, background switching, navigation, tools).
- `assets/day.mp4`, `assets/night.mp4` — background video assets (provide your own in `assets/`).
